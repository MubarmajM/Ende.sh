
# Ende.sh

A Bash Script for Encoding/Decoding strings.




## Installation

Install my-project with npm

```bash
  git clone https://github.com/MubarmajM/Ende.sh.git
  cd Ende.sh/
  pip install -r requirements.txt 
```
Use Ende.sh for the original Encoder/Decoder.              
```bash
chmod +x Ende.sh
./Ende.sh
```

You can also use Ende.py for easier usage.
```bash
python3 Ende.py
```



## Usage/Examples

For Ende.sh:

```bash
./Ende.sh -d base64 "HelloWorld" # For decoding
```
Type ```Ende.sh -h``` for help.
